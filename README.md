# Portfolio | Lindoberto Fernandes

Site do meu portfólio desenvolvido para apresentar projetos e competências como desenvolvedor full-stack.

O objetivo é mostrar alguns dos trabalhos mais relevantes em desenvolvimento web e automação que fiz.

---

## Tecnologias

* React + TypeScript (Vite)
* Tailwind CSS
* Framer Motion
* Lucide React

---

## Estrutura

* `src/components`: componentes reutilizáveis
* `src/data`: dados dos projetos
* `src/styles`: estilos globais

---

## Execução local

```bash
npm install
npm run dev
```

---

## Deploy Automático

Este projeto utiliza **GitHub Actions** para CI/CD. Sempre que um push é feito para a branch `main`, o projeto é automaticamente construído e publicado no GitHub Pages.

Certifique-se de configurar em seu repositório:
`Settings > Pages > Build and deployment > Source: GitHub Actions`
